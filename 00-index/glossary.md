## Termos

- **Wedge**: recorte inicial estreito (1–2 rotinas) com ROI claro para entrar no mercado sem dispersão.
- **HIL (Human-in-the-loop)**: etapa onde ações sensíveis passam por aprovação humana antes de executar.
- **Tenant**: instância lógica de cliente; todos os dados e operações são filtrados por `tenant_id`.
- **RAG (Retrieval-Augmented Generation)**: recuperar contexto relevante e gerar resposta com citações.
- **Guardrails**: mecanismos para limitar ou validar saídas da IA (schemas, filtros, aprovação).
- **LLMOps**: práticas para operar LLMs em produção (observabilidade, qualidade, custo, segurança).
- **SLO**: objetivo de nível de serviço (ex.: latência p95 ≤ 4s, custo ≤ R$0,50/execução).
- **Rollback**: reversão de ação ou deploy para estado anterior seguro.
- **Auto-eval**: avaliação automatizada de respostas do modelo com rubricas e datasets de teste.
- **Consentimento/Legítimo Interesse**: bases legais para tratamento de dados (LGPD) em contexto B2B/outbound.
